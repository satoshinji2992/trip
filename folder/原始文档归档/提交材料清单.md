## 2. 源代码

提交仓库源码即可，主要目录：

```text
backend/     Flask后端、数据库模型、API、算法服务、初始化脚本
frontend/    React前端页面、API封装、地图和交互组件
wan2.2/      AIGC视频生成服务器代码
deploy/      Gunicorn和Nginx部署配置
scripts/     发布包构建脚本
```

本地运行数据如 `backend/trip.db`、`backend/uploads/`、`backend/whoosh_index/` 不建议直接纳入Git版本管理，但可放入课程提交的可运行压缩包。

## 3. 可执行代码

执行：

```bash
./scripts/build_release.sh
```

生成：

```text
release/trip-runnable/
release/trip-runnable.zip
```

发布包包含后端、前端、数据库、文档、部署配置和启动说明。具体见 `可执行代码打包说明.md`。
