# 个性化旅游系统 - 前后端对接文档（API接口文档）

## 一、通用约定

### 1.1 基础信息
- **后端地址**: `http://localhost:5001`
- **前端代理**: 开发环境下Vite将 `/api` 前缀请求代理到后端
- **数据格式**: JSON
- **认证方式**: Cookie Session（Flask-Login）

### 1.2 统一响应格式
```json
{
  "code": 200,
  "message": "操作成功",
  "data": { ... }
}
```
- `code >= 400` 表示错误
- `401` 表示未登录

## 二、用户认证接口 (`/api/auth`)

**请求体**:
```json
{
  "username": "zhangsan",
  "password": "123456",
  "email": "zhangsan@example.com",
  "nickname": "张三",
```

**请求体**:
```json
{ "username": "zhangsan", "password": "123456" }
```
**响应**: 用户信息对象

**需要认证**: 是

**需要认证**: 是

```json
{ "nickname": "新昵称", "interests": ["美食"], "email": "new@example.com" }
```

## 三、景区/校园接口 (`/api/scenic`)

### GET `/api/scenic/list` - 景区列表（排序推荐）
**查询参数**:
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| page | int | 否 | 页码，默认1 |
| per_page | int | 否 | 每页数量，默认20 |
| type | string | 否 | scenic/campus |
| category | string | 否 | 分类过滤 |
| sort_by | string | 否 | popularity/rating/mixed，默认mixed |

**响应**:
```json
{
  "items": [{ "id": 1, "name": "故宫博物院", "type": "scenic", "popularity": 50000, "rating": 4.8 }],
  "total": 202, "page": 1, "per_page": 20, "pages": 11
}
```

### GET `/api/scenic/recommend` - 推荐Top-K
**查询参数**: `sort_by`, `top_k`(默认10), `type`

### GET `/api/scenic/search` - 搜索景区
**查询参数**: `q`(必填), `type`, `category`, `sort_by`

### GET `/api/scenic/{id}` - 景区详情
**响应**: 景区完整信息 + buildings列表 + facilities列表

### GET `/api/scenic/categories` - 获取分类列表
### GET `/api/scenic/types` - 获取类型列表

## 四、路线规划接口 (`/api/route`)

### POST `/api/route/shortest` - 最短路径（Dijkstra）
**请求体**:
```json
{
  "scenic_id": 1,
  "from_node_id": 1,
  "to_node_id": 10,
  "strategy": "distance",
  "transport": "walk"
}
```
- `strategy`: `distance`(最短距离) / `time`(最短时间)
- `transport`: `walk`(步行) / `bike`(自行车) / `cart`(电瓶车)

**响应**:
```json
{
  "path": [1, 3, 5, 10],
  "path_details": [{ "node_id": 1, "name": "正门", "x": 100, "y": 200 }, ...],
  "total_cost": 450.5,
  "cost_unit": "米",
  "distance": 450.5,
  "time": 5.4,
  "avg_congestion": 0.32,
  "edge_details": [
    { "road_name": "东华门大街", "distance": 120.0, "time": 1.8, "congestion": 0.72, "actual_speed": 1.4 }
  ],
  "node_count": 4
}
```

### POST `/api/route/multi` - 多点路线规划（TSP）
**请求体**:
```json
{
  "scenic_id": 1,
  "start_node_id": 1,
  "destinations": [5, 8, 12],
  "strategy": "distance",
  "transport": "walk",
  "return_to_start": true
}
```
**响应**:
```json
{
  "ordered_path": [1, 3, 5, 7, 8, 6, 12, 2, 1],
  "visit_order": [5, 8, 12],
  "total_cost": 1200.5,
  "segments": [{ "from_node_id": 1, "to_node_id": 5, "cost": 300.2, "path": [...] }, ...]
}
```

### GET `/api/route/transport-options?scenic_id=1` - 交通工具选项
### GET `/api/route/nodes?scenic_id=1` - 道路图节点
### GET `/api/route/edges?scenic_id=1` - 道路图边
### GET `/api/route/map-data?scenic_id=1` - 完整地图数据

## 五、场所查询接口 (`/api/facility`)

### GET `/api/facility/nearby` - 查找附近设施（路径距离排序）
**查询参数**:
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| scenic_id | int | 是 | 景区ID |
| node_id | int | 是 | 当前位置节点ID |
| type | string | 否 | 设施类型过滤 |
| max_distance | float | 否 | 最大距离(米)，默认1000 |

**说明**: 返回结果按**实际可达路径距离**排序（非直线距离）

### GET `/api/facility/search` - 搜索设施
**查询参数**: `scenic_id`(必填), `q`(必填), `node_id`(可选)

### GET `/api/facility/types` - 设施类型列表
### GET `/api/facility/by-category` - 按类别过滤

## 六、旅游日记接口 (`/api/diary`)

### POST `/api/diary/create` - 创建日记（需认证）
**请求体**:
```json
{
  "title": "故宫一日游",
  "content": "今天参观了故宫...",
  "scenic_id": 1,
  "destination": "故宫博物院",
  "tags": ["历史文化", "北京"],
  "is_public": true
}
```
**说明**: 内容使用zlib无损压缩存储，自动索引到全文搜索

### PUT `/api/diary/update/{id}` - 更新日记（需认证）
### DELETE `/api/diary/delete/{id}` - 删除日记（需认证）
### GET `/api/diary/{id}` - 日记详情（自动增加浏览量）

**响应包含**:
- `content` - 解压后的内容
- `compression_ratio` - 压缩率
- `images` - 图片列表
- `videos` - 已生成的AIGC旅游动画列表
- `comments` - 评论列表

### GET `/api/diary/my` - 我的日记（需认证）
### GET `/api/diary/public` - 所有公开日记（推荐排序）
**查询参数**: `page`, `per_page`, `sort_by`(popularity/rating/mixed)

### GET `/api/diary/by-destination` - 按目的地查找日记
**查询参数**: `destination`(必填), `sort_by`

### GET `/api/diary/search` - 搜索日记（模糊查找）
**查询参数**: `q`(必填), `sort_by`

### GET `/api/diary/fulltext-search` - 全文搜索（BM25F）
**查询参数**: `q`(必填), `limit`(默认50)

### POST `/api/diary/{id}/comment` - 添加评论评分（需认证）
**请求体**:
```json
{ "content": "写得很好！", "rating": 4.5 }
```

### POST `/api/diary/{id}/upload-image` - 上传日记图片（需认证，multipart/form-data）
**说明**: 图片保存到 `backend/uploads/diary/{diary_id}/`

### DELETE `/api/diary/{id}/image/{image_id}` - 删除日记图片（需认证）

### POST `/api/diary/{id}/generate-animation` - 生成旅游动画（需认证）
**请求体**:
```json
{
  "image_id": 1,
  "prompt": "Create a 5-second cinematic yet realistic travel video...",
  "size": "832*480",
  "sample_steps": 50,
  "frame_num": 121,
  "seed": -1
}
```
**说明**:
- `size` 支持 `832*480`、`480*832`、`1280*704`、`704*1280`
- `frame_num` 按24fps计算，推荐 `24 * 秒数 + 1`
- 生成结果保存到 `backend/uploads/diary/{diary_id}/animation_*.mp4`

### POST `/api/diary/rebuild-index` - 重建全文搜索索引

## 七、美食推荐接口 (`/api/food`)

### GET `/api/food/recommend` - 美食推荐（Top-K排序）
**查询参数**:
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| scenic_id | int | 是 | 景区ID |
| sort_by | string | 否 | popularity/rating/distance/mixed |
| cuisine | string | 否 | 菜系过滤 |
| top_k | int | 否 | 返回前K个，默认10 |
| node_id | int | 否 | 距离排序时的用户位置 |

### GET `/api/food/search` - 美食搜索（模糊查找）
**查询参数**: `scenic_id`(必填), `q`(必填), `sort_by`, `node_id`

**响应**:
```json
{
  "restaurants": [{ "name": "老王中餐馆", "cuisine": "chinese", "rating": 4.5, "distance": 300, ... }],
  "foods": [{ "name": "红烧肉", "restaurant_name": "老王中餐馆", "price": 38, ... }]
}
```

### GET `/api/food/cuisines` - 菜系列表
### GET `/api/food/restaurant/{id}` - 餐厅详情及菜品
