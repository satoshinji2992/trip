# 个性化旅游系统课程设计提交材料说明

本目录按课程要求将提交内容分为 1-4 四个独立文件夹。

## 课程设计报告表

包含：

- `课程设计报告表-个性化旅游系统.docx`：已根据空白模板整理的课程设计报告表。

## 源代码

包含：

- `backend/`：Flask 后端源码、模型、路由、算法服务、初始化脚本。
- `frontend/`：React 前端源码、页面组件、API 封装。
- `wan2.2_traval_fintuned/`：AIGC 视频生成服务器源码。
- `deploy/`：Gunicorn 和 Nginx 部署配置。
- `scripts/`：发布包构建脚本。
- `README.md`、`.gitignore`：项目说明和 Git 忽略规则。

该目录不包含 `node_modules`、`venv`、数据库、上传文件、缓存文件和模型权重。

## 可执行代码

包含：

- `trip-runnable.zip`：可运行发布包压缩文件。

推荐启动方式：

```bash
cd trip-runnable
chmod +x setup_and_start_nginx.sh
./setup_and_start_nginx.sh
```

默认访问：

```text
http://localhost:8080
```

该脚本会自动安装依赖、生成 `.env`、初始化数据库、构建前端，并使用 Nginx + Gunicorn 启动系统。
